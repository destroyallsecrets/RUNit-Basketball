# RUNIT BASKETBALL

### OWSE - Outside We Stand Eternally

*The basketball social app for hoopers. Find courts, join games, connect with players. Works offline.*

[![License: MIT](https://img.shields.io/badge/License-MIT-red.svg)](https://opensource.org/licenses/MIT)
[![Deployed on Vercel](https://img.shields.io/badge/Vercel-Deploy-black)](https://runit-basketball.vercel.app)

</div>

---

## Features

### Court Discovery
- Browse basketball courts with detailed info (surface type, skill level, player count)
- Filter by distance, skill level, or activity
- Add and customize your own courts
- Favorite courts for quick access

### Game System
- Create pickup games (3v3, 4v4, 5v5)
- Join existing games at any court
- View all players in a game
- Filter by skill level (Casual, Competitive, Elite)

### Social Feed
- Share your runs and highlights
- Like and comment on posts
- Location-tagged moments

### Player Profiles
- Track your games played and win rate
- Customizable skill level and archetype
- Screenshot-friendly share cards for offline sharing

### Offline-First
- All data stored locally in browser
- Works without internet connection
- PWA installable on mobile

## Tech Stack

- **Frontend:** React + TypeScript + Vite
- **Styling:** Tailwind CSS + shadcn/ui
- **Icons:** Lucide React
- **State:** React Context + localStorage
- **PWA:** Vite PWA Plugin
- **Deploy:** Vercel

---

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
